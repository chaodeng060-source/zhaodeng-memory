# 朝灯的记忆库 - 部署指南 📝

## 这是什么？

一个轻量级的记忆管理系统，让 Claude 能够：
- 保存和读取记忆
- 写日记（带心情）
- 基于遗忘曲线自动管理记忆
- 主动浮现重要/高情绪的记忆

## 记忆分类

| 分类 | 用途 | 会遗忘吗 |
|------|------|----------|
| `core` | 永久信息（身份、偏好） | ❌ 永不 |
| `daily` | 日常事件 | ✅ 3天后衰减 |
| `diary` | 日记 | ❌ 永久 |
| `milestone` | 重要里程碑 | ❌ 永久 |
| `mood` | 情绪状态 | ✅ 7天后衰减 |

## 部署到 Zeabur（推荐，最简单）

### 第一步：准备代码

1. 去 GitHub 创建一个新仓库（比如叫 `my-memory-mcp`）
2. 把这三个文件上传上去：
   - `server.js`
   - `package.json`
   - `zeabur.json`

### 第二步：Zeabur 部署

1. 打开 https://zeabur.com
2. 点 "创建新项目"
3. 选择 "导入项目" → 选你刚创建的 GitHub 仓库
4. Zeabur 会自动检测并部署
5. 部署完成后，点击项目 → "域名" → 生成一个域名（比如 `xxx.zeabur.app`）

### 第三步：添加到 Claude.ai

1. 打开 claude.ai → 设置 → Connectors → 添加自定义连接器
2. 名称：`记忆库`（或任意名字）
3. URL：`https://你的域名.zeabur.app/sse`
4. 认证：None
5. 保存

### 第四步：测试

1. **开新对话**（重要！旧对话看不到新连接器）
2. 跟 Claude 说："帮我保存一条记忆：今天学会了部署 MCP 服务器"
3. Claude 应该会调用 `memory_save` 工具

## 可用的工具

### memory_save - 保存记忆
```
保存一条记忆：今天天气很好
保存一条 milestone 记忆：论文答辩通过了！重要度 10
```

### memory_read - 读取记忆
```
读取我最近的日记
读取所有 core 类型的记忆
```

### memory_search - 搜索
```
搜索关于"论文"的记忆
```

### memory_surface - 主动浮现
```
有什么重要的事情要提醒我吗？
让记忆自己浮现一下
```

### memory_diary - 写日记
```
帮我写今天的日记：今天和朋友出去玩了，很开心！心情 0.8
```

### memory_stats - 统计
```
我的记忆库有多少条记忆？
```

## 迁移数据

如果以后想迁移到鹤见的 Ombre Brain：
1. 数据存在 SQLite 数据库 `memory.db`
2. 可以从 Zeabur 下载
3. 写个小脚本转换格式即可

## 遇到问题？

- 部署失败：检查 GitHub 仓库是否 public
- 连接不上：检查 URL 是否带 `/sse` 后缀
- 工具没显示：**开新对话！**
